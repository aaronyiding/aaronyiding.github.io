MADNet

Overview

  • Metropolitan Advanced Delivery Network (MADNet) is a collaborative WiFi-based mobile data offloading architecture that targets to improve energy efficiency for smartphones. MADNet achieves collaboration by intelligently aggregating the power of cellular operators, WiFi service providers and end-users. It utilizes an energy-aware algorithm to assist the offloading decision for smartphones, which enables them to select the most energy efficient WiFi APs to offload mobile traffic. MADNet project is in collaboration with Deutsche Telekom T-Labs, AT&T and University of Maryland College Park.

NAO

Overview

  • Network-Assisted Offloading (NAO) is a framework that utilizes network assistance to achieve efficient IP traffic offloading. It enables mobile network operators to actively trigger and guide the traffic offloading through dedicated protocols. NAO supports two offloading modes, namely host-driven mode with DHCPv6 and network-driven mode with Neighbor Discovery. NAO project is in collaboration with Nokia and Broadcom.
